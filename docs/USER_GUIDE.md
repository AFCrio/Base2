## 1. Перший запуск

При першому запуску додаток автоматично:
- Створює базу даних `DutyOrder.db` у папці програми
- Застосовує всі необхідні міграції
- Завантажує початкові дані (пресети боєкомплектів)

**Рекомендований порядок початкової роботи:**
1. Заповнити довідники (звання, посади, особовий склад, зброя, транспорт, локації)
2. Створити шаблон наряду
3. Створювати накази на основі шаблону

## 2. Головне вікно

Головне вікно — центр управління додатком.

### Елементи інтерфейсу:
- **Вибір шаблону** (випадаючий список) — оберіть активний шаблон
- **Таблиця наказів** — список наказів, створених з обраного шаблону
- **Попередження** — повідомлення про накази, створені зі старої версії шаблону

### Операції з шаблонами:
| Кнопка | Дія |
|--------|-----|
| **+ Шаблон** | Створити новий шаблон (введіть назву) |
| **Редагувати шаблон** | Відкрити редактор структури шаблону |
| **Видалити шаблон** | Видалити шаблон (лише якщо немає створених наказів) |

### Операції з наказами:
| Кнопка | Дія |
|--------|-----|
| **+ Наказ** | Створити новий наказ з обраного шаблону |
| **Редагувати** | Відкрити наказ для редагування (або подвійний клік) |
| **Видалити** | Видалити наказ з усіма призначеннями |
| **Оновити** | Оновити таблицю наказів |

### Додатково:
| Кнопка | Дія |
|--------|-----|
| **Довідники** | Відкрити управління довідниками |
| **Налаштування** | Відкрити налаштування додатку |

## 3. Робота з довідниками

Форма довідників містить 7 вкладок. Заповніть їх перед створенням шаблонів та наказів.

### 3.1 Локації
Місця зберігання зброї та розташування постів.
- **Назва** — назва локації (наприклад: «Пост №1», «Склад зброї»)
- **Адреса** — адреса (необов'язково)

> 💡 Локація прив'язується до вузлів шаблону для фільтрації доступної зброї.

### 3.2 Звання
Військові звання з рівнем для сортування.
- **Назва** — назва звання (солдат, сержант, лейтенант тощо)
- **Рівень** — число для сортування (1 = солдат, 11 = майор)

### 3.3 Посади
Військові посади.
- **Назва** — назва посади (стрілець, командир відділення, водій тощо)

### 3.4 Особовий склад
Військовослужбовці підрозділу.
- **Прізвище** — ВЕЛИКИМИ літерами
- **Ім'я** — ім'я (для ініціалів)
- **По батькові** — по батькові (для ініціалів, може бути порожнім)
- **Звання** — обрати зі списку
- **Посада** — обрати зі списку

### 3.5 Зброя
Зброя підрозділу.
- **Тип** — тип зброї (АК-74, ПМ, АКМ тощо)
- **Номер** — серійний номер
- **Локація зберігання** — де зберігається (для фільтрації)
- **Закріплена за** — особа, за якою закріплена зброя

> 💡 Якщо зброя закріплена за особою, вона автоматично пропонується при призначенні цієї особи.

### 3.6 Транспорт
Транспортні засоби.
- **Назва** — марка/модель
- **Номер** — державний номер
- **Тип** — легковий, вантажний, санітарний тощо

### 3.7 Боєкомплекти
Типові набори боєкомплектів для типів зброї.
- **Тип зброї** — назва типу (має відповідати типу в довіднику зброї)
- **Тип набоїв** — калібр (9 мм, 5,45 мм тощо)
- **Кількість** — стандартна кількість набоїв

> 💡 При призначенні зброї, тип та кількість набоїв заповнюються автоматично.

### Загальні дії для всіх вкладок:
- **Фільтр** — поле пошуку над таблицею (фільтрує записи по першим літерам)
- **Додати** — створити новий запис
- **Редагувати** — змінити обраний запис (або подвійний клік)
- **Видалити** — видалити запис (заборонено, якщо є залежні дані)

## 4. Створення шаблону

Шаблон визначає структуру добового наряду — які розділи, посади та групи він містить.

### Кроки:
1. У головному вікні натисніть **[+ Шаблон]**
2. Введіть назву шаблону (наприклад: «Основний добовий наряд»)
3. Автоматично відкриється **Редактор шаблонів**

---

## 5. Редагування шаблону

### 5.1 Структура шаблону
Шаблон — це **дерево вузлів**, де кожен вузол має свій тип:

| Тип вузла | Іконка | Призначення | Приклад |
|-----------|--------|-------------|---------|
| **Заголовок розділу** | 📋 | Організаційний заголовок | «1. Нарядна служба» |
| **Проста посада** | 👤 | Одна людина | «Черговий містечка» |
| **Водій** | 🚗 | Посада з транспортом | «Водій чергової машини» |
| **Група (рядок)** | 👥 | Кілька осіб в рядок | «Навідники: ос1, ос2, ос3» |
| **Група (список)** | 👥 | Кілька осіб списком | Кожна особа — окремий підпункт |
| **Часовий діапазон** | 🕐 | Зміна/вахта | «Зміна 1 (08:00-20:00)» |
| **Вогнева група** | 🔥 | Спеціальний розділ | Пожежний розрахунок |

### 5.2 Додавання вузлів

1. **Правий клік** на вузлі дерева → контекстне меню
2. Оберіть дію:
   - **⬇ Додати дочірній** — створити підпункт обраного вузла
   - **↔ Додати сусідній** — створити вузол на тому ж рівні
3. Оберіть тип вузла у діалозі
4. Заповніть параметри на правій панелі

### 5.3 Шаблон посади (плейсхолдери)

У полі «Шаблон посади» використовуйте **плейсхолдери** — мітки в фігурних дужках, які автоматично замінюються на реальні дані при призначенні:

**Приклад шаблону:**
```
Черговий містечка – {Rank} {LastName} {Initials}
```
**Результат після призначення:**
```
Черговий містечка – солдат ПЕТРЕНКО П.В.
```

**Доступні плейсхолдери:**

| Плейсхолдер | Опис | Приклад |
|-------------|------|---------|
| `{Rank}` | Звання | солдат |
| `{LastName}` | Прізвище | ПЕТРЕНКО |
| `{Initials}` | Ініціали | П.В. |
| `{FullName}` | Прізвище + ініціали | ПЕТРЕНКО П.В. |
| `{Position}` | Посада | стрілець |
| `{WeaponType}` | Тип зброї | АК-74 |
| `{WeaponNumber}` | Номер зброї | 174507 |
| `{AmmoType}` | Тип набоїв | 5,45 мм |
| `{AmmoCount}` | Кількість набоїв | 120 |
| `{VehicleName}` | Назва транспорту | УАЗ |
| `{VehicleNumber}` | Номер транспорту | АА1234ВВ |
| `{OrderPeriod}` | Період наряду | з 19 по 20 січня 2026 року |
| `{CommanderInfo}` | Інформація про командира | майор ІВАНОВ І.П. |

> 💡 Якщо плейсхолдер не заповнений — відображається `___` (прочерк).

### 5.4 Прапорці вузла

- **☑ Зброя** — позиція потребує видачі зброї
- **☑ Боєкомплект** — позиція потребує видачі набоїв
- **☑ Транспорт** — позиція потребує транспорту

> 💡 Якщо прапорець встановлено, але в шаблоні немає відповідного плейсхолдера — дані автоматично додаються в кінець рядка.

### 5.5 Максимальна кількість призначень

- **1** — для простих посад (один чергуючий)
- **0** — для груп (необмежена кількість)
- **N** — конкретний ліміт для групи

### 5.6 Переміщення вузлів

- **Drag & Drop** — перетягніть вузол мишкою на нове місце
- **▲ Вгору / ▼ Вниз** — контекстне меню для зміни порядку
- **🗑️ Видалити** — видалити вузол (з усіма дочірніми)

### 5.7 Збереження

Натисніть **[💾 Зберегти]**. Версія шаблону автоматично збільшується, а зміни записуються у журнал.

## 6. Створення наказу

### Кроки:
1. У головному вікні оберіть шаблон зі списку
2. Натисніть **[+ Наказ]**
3. Відкриється **форма створення наказу**
4. Заповніть метадані:
   - **Номер наказу** — унікальний ідентифікатор (наприклад: А4463/19-01-2026)
   - **Дата** — дата наказу
   - **Початок** — дата та час початку наряду
   - **Кінець** — дата та час закінчення наряду
   - **Командир** — ПІБ та звання командира
5. Якщо в шаблоні є **часові зміни** — налаштуйте час кожної зміни
6. Призначте особовий склад (див. розділ 7)
7. Натисніть **[💾 Зберегти]**

### Кольорова індикація в дереві:
- 🟢 **Зелений** — позиція повністю заповнена
- 🔴 **Червоний** — позиція не заповнена
- 🟠 **Помаранчевий** — група частково заповнена

### Попередній перегляд
Праворуч відображається **текстовий попередній перегляд** наказу з усіма підставленими даними.

---

## 7. Призначення особового складу

### Кроки:
1. У формі наказу **подвійний клік** на вузол дерева (або ПКМ → «Додати призначення»)
2. Відкриється **форма призначення**

### Вибір особи:
1. Введіть перші літери **прізвища** у поле пошуку
2. Таблиця автоматично фільтрується
3. Оберіть особу кліком або клавішами ↑↓ + Enter
4. Відображаються лише **непризначені** особи (вже зайняті не показуються)

### Зброя:
- **Закріплена зброя** — автоматично пропонується зброя, закріплена за обраною особою
- **Інша зброя** — вибір з доступних (фільтрується за локацією вузла)

### Боєкомплект:
- **Тип набоїв** та **кількість** заповнюються автоматично на основі типу зброї (з пресетів)
- Можна змінити вручну

### Транспорт:
- Обрати з доступних транспортних засобів (ще не призначені в цьому наказі)

### Додавання нової особи:
Якщо потрібної особи немає у списку — натисніть **[+ Додати особу]** для швидкого створення без закриття форми.

---

## 8. Редагування наказу

1. У головному вікні **подвійний клік** на наказі (або кнопка **[Редагувати]**)
2. Відкриється форма з усіма збереженими даними
3. Можна:
   - Змінити метадані (номер, дата, командир)
   - Додати/видалити призначення (ПКМ на вузлі → «Видалити призначення»)
   - Змінити часові діапазони
4. Натисніть **[💾 Зберегти]** для збереження змін

---

## 9. Налаштування

У формі налаштувань доступні:
- **Звання за замовчуванням** — для нових осіб
- **Посада за замовчуванням** — для нових осіб

---

## ⚠️ Важливі примітки

1. **Версіонування шаблонів**: При редагуванні шаблону його версія збільшується. Накази, створені зі старої версії, позначаються попередженням у головному вікні.

2. **Унікальність**: Номер наказу, номер зброї, номер транспорту, назви звань та посад — мають бути унікальними.

3. **Захист від видалення**: Не можна видалити звання, посаду або особу, якщо вони використовуються в наказах. Спочатку видаліть залежні записи.

4. **Автонумерація**: Номери пунктів (1, 1.1, 1.2 тощо) генеруються автоматично на основі структури дерева.

5. **Збережений текст**: Кожне призначення зберігає «заморожений» текст (RenderedLine). Це гарантує, що текст наказу не зміниться при подальшому редагуванні довідників.
