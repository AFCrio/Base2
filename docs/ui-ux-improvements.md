# Пропозиції щодо покращення UI/UX

## Критичні (впливають на продуктивність щодня)

### 1. Статус-бар із зворотним зв'язком
**Проблема:** Немає підтвердження при збереженні. Користувач не знає, чи збереглися зміни.  
**Рішення:** Додати `StatusStrip` з текстом стану та іконкою (✅ Збережено / ⚠️ Помилка).

### 2. Підтвердження незбережених змін
**Проблема:** При закритті `TemplateEditorForm` або `OrderCreatorForm` із незбереженими змінами — дані губляться без попередження.  
**Рішення:** При закритті форми перевіряти `isDirty` → показувати діалог «Є незбережені зміни. Зберегти?».

### 3. Клавіатурні скорочення
**Проблема:** Відсутні hot keys.  
**Рекомендовані:**
- `Ctrl+S` — зберегти
- `F5` — оновити список
- `Delete` — видалити вибраний запис (з підтвердженням)
- `Escape` — закрити форму / скасувати
- `Insert` — додати новий запис

### 4. Живий пошук у AssignmentForm
**Проблема:** Пошук по прізвищу потребує підтвердження кнопкою.  
**Рішення:** Фільтрація при введенні (`TextChanged` з debounce 300мс).

### 5. Валідація дат у наряді
**Проблема:** Немає перевірки `EndDateTime > StartDateTime`.  
**Рішення:** Вивести попередження у формі при порушенні логіки дат.

---

## Важливі (впливають на зручність)

### 6. Пагінація / серверна фільтрація у великих довідниках
**Проблема:** При 1000+ осіб `DataGridView` гальмує.  
**Рішення:** Завантажувати лише відфільтровані записи з БД (передавати `filter` до сервісу), а не всі одразу.

### 7. Замінити ComboBox шаблонів на ListBox із пошуком
**Проблема:** При > 20 шаблонах ComboBox незручний.  
**Рішення:** `ListBox` зліва + поле фільтра → сортування за датою створення.

### 8. Сортування колонок у DataGridView
**Проблема:** Колонки не сортуються натисканням заголовка.  
**Рішення:** Увімкнути `SortMode = Automatic` + реалізувати сортування через сервіс.

### 9. Навігація по довідниках
**Проблема:** 8 вкладок у ReferenceForm важко знаходити.  
**Рішення:** Додати вертикальну навігаційну панель (`TreeView` або `ListBox`) зліва замість вкладок.

### 10. «Хто зайняв» зброю / транспорт
**Проблема:** Список зброї у AssignmentForm не показує, за ким вона закріплена.  
**Рішення:** Додати стовпець «Закріплено за» у таблицю зброї.

### 11. Прогрес заповнення наряду
**Проблема:** Немає індикатора, скільки позицій вже заповнено.  
**Рішення:** Показати `X / Y призначено` у заголовку або статус-барі.

### 12. Колірна індикація у дереві наряду
**Рішення:**
- 🟢 Зелений — позиція повністю заповнена
- 🟡 Жовтий — заповнена частково (є місця)
- ⚪ Сірий — порожньо

## Покращення UX (середній пріоритет)

### 13. Скасувати (Undo) у редакторі шаблонів
Стек операцій: переміщення вузлів, видалення, редагування назв.

### 14. Останні відкриті наряди на головному екрані
Список 5 останніх нарядів для швидкого доступу (`MRU list`).

### 15. Друк / Експорт у PDF або DOCX
Функція формування підсумкового документу наряду (таблиця для підпису).

### 16. Тема оформлення (Dark mode)
Актуально для нічних чергувань. `ColorScheme` або системна тема Windows.

### 17. Підказки (tooltips) на всіх кнопках
Для нових користувачів корисні підказки при наведенні миші.

### 18. Пошук по всьому наряду
Пошук особи або зброї у всіх призначеннях поточного наряду.

## Дрібні виправлення

| № | Проблема | Рішення |
|---|---------|---------|
| 19 | Помилки валідації показуються через `MessageBox` | Замінити на inline-підказки або `ErrorProvider` |
| 20 | Ширина колонок DataGridView скидається при кожному відкритті | Зберігати ширину у `AppSettings` |
| 21 | `Form1.cs` — залишковий файл від шаблону | Видалити |
| 22 | Відсутній `Tab order` у формах | Налаштувати правильний порядок Tab |
| 23 | Немає фокусу на першому полі при відкритті форм | Встановити `ActiveControl` у конструкторі |
