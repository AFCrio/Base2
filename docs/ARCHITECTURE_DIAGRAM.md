## 📝 Потік роботи з шаблонами

```
┌─────────────────────────────────────────────────────────────┐
│              РЕДАКТОР ШАБЛОНІВ (TemplateEditorForm)          │
│                                                              │
│  Дерево структури:              Панель редагування:          │
│  ┌───────────────────────┐      ┌────────────────────────┐  │
│  │ 📋 1. Нарядна служба  │      │ Тип: [Заголовок     ▼] │  │
│  │  ├ 👤 1.1 Черговий     │      │                        │  │
│  │  ├ 👤 1.2 Пом.чергового│      │ Шаблон посади:         │  │
│  │  ├ 👥 1.3 Навідники    │      │ [Черговий – {Rank}     │  │
│  │  │  ├ 👤 Навідник 1    │      │  {LastName} {Initials}]│  │
│  │  │  └ 👤 Навідник 2    │      │                        │  │
│  │  └ 🕐 1.4 Зміна 1     │      │ ☑ Зброя ☑ БК ☐ Транс. │  │
│  │     ├ 👤 Вартовий 1    │      │ Макс. призначень: [1]  │  │
│  │     └ 👤 Вартовий 2    │      │                        │  │
│  │ 📋 2. Вогнева група    │      │ Локація: [Пост №1   ▼] │  │
│  │  └ ...                 │      │                        │  │
│  └────────────────────────┘      │     [💾 Зберегти]      │  │
│                                   └────────────────────────┘  │
│  Контекстне меню (ПКМ):                                      │
│  ⬇ Додати дочірній │ ↔ Додати сусідній │ 🗑️ Видалити        │
│  ▲ Вгору │ ▼ Вниз │ Drag&Drop для переміщення               │
└──────────────────────────────────────────────────────────────┘

Послідовність створення шаблону:

  1. Натиснути [+ Шаблон] у MainForm
         │
         ▼
  2. Ввести назву шаблону → створюється порожній шаблон
         │
         ▼
  3. Відкривається TemplateEditorForm
         │
         ▼
  4. ПКМ → "Додати дочірній" → обрати тип вузла
         │
         ├─ SectionHeader (Заголовок розділу)
         ├─ SimplePosition (Одна посада)
         ├─ DriverPosition (Водій)
         ├─ GroupInline (Група в рядок)
         ├─ GroupNested (Група з підпунктами)
         ├─ TimeRange (Часова зміна)
         └─ FireGroup... (Вогнева група)
         │
         ▼
  5. Заповнити шаблон посади з плейсхолдерами
         │
         ▼
  6. Повторити 4-5 для всіх позицій → Зберегти
```

  - Поле фільтрації (пошук за першими літерами)

  - Підтвердження перед видаленням

## 🔄 Життєвий цикл даних

```
                    Довідники
                    (рідко змінюються)
                         │
    ┌────────────────────┼────────────────────┐
    │                    │                    │
    ▼                    ▼                    ▼
┌────────┐        ┌──────────┐        ┌──────────┐
│Звання  │        │  Особи   │        │  Зброя   │
│Посади  │        │(персонал)│        │Транспорт │
│Локації │        │          │        │Пресети БК│
└────┬───┘        └────┬─────┘        └────┬─────┘
     │                 │                   │
     └────────┬────────┴───────────────────┘
              │
              ▼
      ┌───────────────┐
      │   ШАБЛОН      │  ← Структура (дерево вузлів)
      │   (Template)  │     Версіонування
      │   v.3         │     Журнал змін
      └───────┬───────┘
              │  клонування
              ▼
      ┌───────────────┐
      │   НАКАЗ       │  ← Конкретний документ
      │   (Order)     │     Метадані (номер, дата)
      │   A4463/...   │     Часові діапазони (зміни)
      └───────┬───────┘
              │  призначення
              ▼
      ┌───────────────┐
      │ ПРИЗНАЧЕННЯ   │  ← Особа + Зброя + БК + Транспорт
      │ (Assignment)  │     Відрендерений текст (кеш)
      └───────────────┘


Версіонування шаблонів:

  Шаблон v.1 ──► Наказ A001 (створено з v.1)
       │
  Редагування шаблону
       │
  Шаблон v.2 ──► Наказ A002 (створено з v.2)
       │
       └──► ⚠️ Наказ A001 позначається як "застарілий"
            (створений зі старої версії шаблону)
```

## 🏗 Діаграма залежностей компонентів

```
┌───────────────────────────────────────────────────────────────┐
│                        UI (Forms)                             │
│                                                               │
│  MainForm ──────┬──► TemplateEditorForm                      │
│                 ├──► OrderCreatorForm ──► AssignmentForm      │
│                 ├──► ReferenceForm                            │
│                 └──► SettingsForm                             │
└───────────────────────────┬───────────────────────────────────┘
                            │ викликає
                            ▼
┌───────────────────────────────────────────────────────────────┐
│                    Services (Бізнес-логіка)                    │
│                                                               │
│  OrderService ──────► AppDbContext                            │
│  TemplateRenderer ──► (статичний, без залежностей)            │
└───────────────────────────┬───────────────────────────────────┘
                            │ використовує
                            ▼
┌───────────────────────────────────────────────────────────────┐
│                    Data (EF Core)                              │
│                                                               │
│  AppDbContext ──► SQLite (DutyOrder.db)                       │
│  14 DbSet<T> (моделі)                                        │
│  13 міграцій                                                  │
└───────────────────────────────────────────────────────────────┘
```
