# Base2 — Добовий наряд

**Система управління добовими нарядами** — десктопне Windows Forms додаток для формування, редагування та управління добовими нарядами військового підрозділу.

## 📋 Опис

Додаток дозволяє:
- Створювати **шаблони** добових нарядів з ієрархічною структурою (розділи, посади, групи, зміни)
- Генерувати **накази** на основі шаблонів з автоматичною нумерацією пунктів
- **Призначати особовий склад** на посади з автоматичним підбором зброї, боєкомплекту та транспорту
- Керувати **довідниками** (особовий склад, зброя, транспорт, звання, посади, локації)
- Відстежувати **версії** шаблонів та виявляти застарілі накази

## 🚀 Запуск

### Вимоги
- .NET 10.0 SDK
- Windows 10/11

```bash
dotnet restore
dotnet build
dotnet run
```

База даних `DutyOrder.db` створюється автоматично при першому запуску з усіма необхідними міграціями.

## 📊 Модель даних

### Довідники
- **Rank** — військові звання (солдат, сержант, майор...) з рівнем сортування
- **Position** — посади (черговий, помічник чергового...)
- **Person** — особовий склад (прізвище, ім'я, звання, посада, закріплена зброя)
- **Weapon** — зброя (тип, номер, локація зберігання, закріплена за особою)
- **Vehicle** — транспорт (назва, номер, тип)
- **Location** — локації (для фільтрації зброї за місцем зберігання)
- **WeaponAmmoPreset** — типові боєкомплекти для типів зброї

### Робочі сутності
- **DutyTemplate** — шаблон наряду (версіонований, з журналом змін)
- **DutySectionNode** — ієрархічний вузол структури (секція/посада/група/зміна)
- **DutyOrder** — конкретний наказ, створений з шаблону
- **DutyAssignment** — призначення особи на посаду (з зброєю, БК, транспортом)
- **DutyTimeRange** — часовий діапазон зміни

### Типи вузлів (NodeType)
| Тип | Опис |
|-----|------|
| `SectionHeader` | Заголовок розділу |
| `SimplePosition` | Одна посада (1 особа) |
| `DriverPosition` | Водій з транспортом |
| `GroupInline` | Група в один рядок |
| `GroupNested` | Група з підпунктами |
| `TimeRange` | Часова зміна |
| `MedicalPosition` | Медична посада |
| `FireGroupSection` | Розділ вогневої групи |
| `FireGroupLocation` | Вогнева група за локацією |
| `FireGroupInline` | Вогнева група в рядок |

## 🔧 Система шаблонів

Шаблони використовують плейсхолдери для динамічної підстановки:

| Плейсхолдер | Значення |
|-------------|---------|
| `{Rank}` | Звання особи |
| `{LastName}` | Прізвище |
| `{Initials}` | Ініціали (В.М.) |
| `{FullName}` | Повне ім'я |
| `{Position}` | Посада |
| `{WeaponType}` | Тип зброї |
| `{WeaponNumber}` | Номер зброї |
| `{AmmoType}` | Тип набоїв |
| `{AmmoCount}` | Кількість набоїв |
| `{VehicleName}` | Назва транспорту |
| `{VehicleNumber}` | Номер транспорту |
| `{OrderPeriod}` | Період наряду |
| `{CommanderInfo}` | Інформація про командира |
| `{TimeLabel}` | Назва зміни |

Приклад: `"Черговий – {Rank} {LastName} {Initials}"` → `"Черговий – солдат ПЕТРЕНКО П.В."`
